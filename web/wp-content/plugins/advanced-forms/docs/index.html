Advanced Forms

Create flexible and developer-friendly forms using the power of Advanced Custom Fields

Download Advanced Forms
Version 1.0.1

Create your form

Advanced Forms comes with an intuitive admin interface allowing you to create forms and easily configure email notifications, display settings, entry creation and much more.

Developers can create their forms fully programmatically allowing for easy integration with themes and plugins.

Add your form fields

Use the standard ACF interface to create field groups and add fields. Simply set up a location rule to add them to your form.

Developers can of course use the functions included with ACF to configure their field groups.

Display your form

Either display your form anywhere using a simple shortcode or include it in your theme/plugin with a function call.

Forms are displayed using minimum styling enabling easy integration with any site.
