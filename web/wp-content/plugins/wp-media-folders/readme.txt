== Description ==

WP media Folders is a WordPress plugin that media management to implement a real folder structure.
A new field is added on the media manager to allow renaming the full path and name of all medias.
The old media url will be also replaced in the posts table.
Other database tables and columns can be selected in the plugin settings.
